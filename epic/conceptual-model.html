Relations

  • One user can create many posts (1 to n)
  • One user can create many comments (1 to n)
  • One user can create many upvotes (1 to n)
  • One user can create many downvotes (1 to n)
  • One post can have many comments (1 to n)
  • One post can have many upvotes (1 to n)
  • One post can have many downvotes (1 to n)
  • One comment can have many upvotes (1 to n)
  • One comment can have many downvotes (1 to n)
  • One comment can have many comments (1 to n)
  • Many posts can be upvoted by many users (m to n)
  • Many comments can be upvoted by many users (m to n)
  • Many posts can be downvoted by many users (m to n)
  • Many comments can be downvoted by many users (m to n)
